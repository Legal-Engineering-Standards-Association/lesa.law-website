# LESO Landing Page

This repository contains the landing page for the LESA project.  

It contains the code for the landing page for the Legal Engineering Stardards Association (aka LESA).

The website is deployed at https://lesa.law

---

## Setup & Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bpetes/deprecated-leso-landingpage.git
   cd deprecated-leso-landingpage
   ```

2. Install dependencies (if applicable):

   ```bash
   npm install
   # or
   yarn install
   ```

3. Build the assets / bundle (if needed):

   ```bash
   npm run build
   # or
   yarn build
   ```

4. Serve locally (for preview):

   ```bash
   npm start
   # or
   yarn start
   ```

   Alternatively, open `index.html` directly in a browser (if no build step is needed).

---

## Usage

Once built, deploy the static output (e.g. `dist/` or root folder) to any static web host (Netlify, GitHub Pages, S3, etc.).
